A coffee-inspired colour theme family for VS Code — espresso browns, latte-foam
backgrounds, and dusty pastel accents. Three roasts, one pot.

## The roasts

### ☕ Dark Roast — warm · darkest · bold
A deep, near-black espresso background with saturated amber, rose, and lavender
accents that really pop.

![Dark Roast preview](https://raw.githubusercontent.com/matdoidge/morning-brew/main/assets/dark-roast.png)

### ☕ Mocha — warm · dark · bold
The same bold beans on a rich, softer coffee background.

![Mocha preview](https://raw.githubusercontent.com/matdoidge/morning-brew/main/assets/mocha.png)

### ☕ Flat White — muted · subtle · light
Soft foam backgrounds with gently desaturated pastels; low-contrast and easy on
the eyes.

![Flat White preview](https://raw.githubusercontent.com/matdoidge/morning-brew/main/assets/flat-white.png)

## Install

Install from the **Visual Studio Marketplace**:

- In VS Code, open **Extensions** (`⇧⌘X` / `Ctrl+Shift+X`), search **“Morning Brew”**, and click **Install**.
- Or from the command line:

  ```sh
  code --install-extension artfuldoidger.morning-brew
  ```

Then pick a roast: **`⌘K ⌘T`** (`Ctrl+K Ctrl+T`) → *Morning Brew — Dark Roast / Mocha / Flat White*.

## Palette

| Role       | Dark Roast / Mocha | Flat White |
| ---------- | ------------------ | ---------- |
| Background | `#170E08` / `#271A12` | `#F5EEE1` |
| Foreground | `#ECDCC4` / `#E9D8BF` | `#504336` |
| Keywords   | `#E0728F`          | `#A66A80`  |
| Strings    | `#9BB86B` (sage)   | `#66814A`  |
| Functions  | `#E0A85B` (amber)  | `#56778C`  |
| Types      | `#C79BE0` (lavender)| `#8271A0` |
| Numbers    | `#E08A4E` (caramel)| `#A9805A`  |
| Comments   | `#8A7259`          | `#AC9C8A`  |

All three share the same beans — rose, sage, amber, lavender, caramel — so they
read as one family. Bracket-pair colours, inlay hints, and terminal ANSI are all
tuned to match.
