# Morning Brew ☕

A coffee-inspired colour theme family for VS Code — espresso browns, latte-foam
backgrounds, and dusty pastel accents.

Two roasts, one pot:

- **Morning Brew — Dark Roast** — warm, dark, and bold. A deep espresso
  background with saturated amber, rose, and lavender accents that pop.
- **Morning Brew — Flat White** — muted, subtle, and light. Soft foam
  backgrounds with gently desaturated pastels; low-contrast and easy on the eyes.

## Install (local / development)

This extension isn't published to the Marketplace yet. To use it locally:

```sh
# Symlink (or copy) the folder into your VS Code extensions directory
ln -s "$(pwd)" ~/.vscode/extensions/morning-brew
```

Then reload VS Code and pick a theme via **⌘K ⌘T** → *Morning Brew — Dark Roast*
or *Morning Brew — Flat White*.

> Using Cursor? Swap `~/.vscode/extensions` for `~/.cursor/extensions`.
